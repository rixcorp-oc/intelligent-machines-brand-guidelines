Callout & Disclosure

Callouts surface important guidance inline. Disclosures collapse secondary content to reduce visual noise. Both reduce cognitive load without hiding critical information.

Callout variants

Tip — Keep each agent focused on a single task. Composing several small agents produces more reliable results than one large agent attempting everything.
Warning — Deleting a workflow also removes all associated run history. This cannot be undone.
Note — Workflow runs are billed per task completion, not per execution. An agent that completes 0 tasks in a run incurs no charge.
