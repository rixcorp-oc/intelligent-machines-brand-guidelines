Gradients

The teal → blue → violet gradient is the brand's primary visual motif. Reserved for CTAs, flow lines, and the logo mark. Never used on surfaces or backgrounds.

Primary — teal → blue → violet · linear-gradient(135deg, #00D4C4, #2563FF, #7B3CFF)

Flow (animated) — left-to-right sweep used on active workflow connectors and loading states

Start building →

Primary CTA button — gradient applied via background:var(--gradient-primary)

tokens.css

--gradient-primary: linear-gradient(135deg, #00D4C4 0%, #2563FF 50%, #7B3CFF 100%);
--gradient-flow: linear-gradient(90deg, #00D4C4, #2563FF, #7B3CFF);
